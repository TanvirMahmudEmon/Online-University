[Pre requisite: CSE 2401]

Objectives:

This course is designed for senior students in Computer Engineering major to introduce them to the components of Internet programming. Course should teach them the design and implementation of different Internet programming components such as html programming, CGI programming, Scripting (server and client sides), Java Programming, JMI. Student should be able to implement different programming assignment and project over the Internet.

Section-A(Mid-term:30 marks)

  • 1. Introduction:

    Introduction to Computer and Internet; Hyper Text Markup Language (HTML): Introduction, Basic concepts, Lists, Tables, Forms, Internal Linking, Image maps, META and FRAMESET Tags.

  • 2. XML and XHTML:

    XML basics; XHTML: Document Structure, Basic Tags, Text Formatting, Character Entities, Lists, Links, Images, Color, Tables, Forms, Image Maps, Meta tags, Framesets.

  • 3. JavaScript:

    Introduction, Control Structures, Functions, Arrays, and Objects.

Section-B(Mid-term:30 marks)

Group-A (20 Marks)

  • 4. Dynamic HTML:

    Introduction, Cascading Style Sheets, Event Handlers, Document Object Model etc.

  • 5. E-commerce:

Group-B (30 Marks)

  • 6. Security :

    Public Key Cryptography, Secure Sockets Layer (SSL), Secure Electronic Transaction (SET).

  • 7. Web Programming:

    WWW database; The creation and updating the web content; Expanding Internet services by adding client-side and server-side processing; Interfacing internet to a database; Querying database using Cold Fusion.

  • 8. Others:

    Overview of server-side technologies, including PHP, ASP etc.

Recommended Books:

  • 1.

    Deitel,and Nieto "Internet & World Wide Web How to program",Third Edition

  • 2.

    Marty Hall and Larry "Core Web programming".Second Edition Brown
